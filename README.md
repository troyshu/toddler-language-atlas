# Toddler Language Atlas

Local-first MVP for a parent-guided toddler language skill atlas.

## App

The web app lives in `app/` and is built with React, TypeScript, and Vite.

```bash
cd app
npm install
npm run dev
```

## Validation

```bash
cd app
npm test
npm run build
npm run lint
```

## GitHub Pages

The site is deployed by `.github/workflows/deploy-pages.yml` from the `app/dist` build artifact.
